## ADDED Requirements

### Requirement: Bun.serve examples
The system SHALL provide examples that demonstrate Bun.serve capabilities.

#### Scenario: basic example
- **WHEN** a developer opens the examples
- **THEN** there is a Bun.serve example that returns a plain Response

#### Scenario: advanced examples
- **WHEN** a developer opens the examples
- **THEN** there are examples for routes/headers, streaming, WebSocket, and TLS proxy mode
