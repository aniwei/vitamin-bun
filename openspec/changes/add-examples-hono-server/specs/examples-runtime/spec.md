## ADDED Requirements

### Requirement: Hono server example
The system SHALL provide a Hono example using Bun.serve in the browser runtime examples.

#### Scenario: Run Hono example
- **WHEN** the developer runs the Hono example
- **THEN** the Bun.serve handler routes requests through a Hono app

### Requirement: Module coverage examples
The system SHALL provide examples that cover HTTP client, net/tls, file IO, and plugin modules.

#### Scenario: Run module examples
- **WHEN** the developer follows the examples README
- **THEN** each module example demonstrates expected runtime behavior
