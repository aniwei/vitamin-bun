# @vitamin-bun/create

项目脚手架工具。

## 安装

```bash
bun add @vitamin-bun/create
```

## 使用

```bash
# 使用 bun create
bun create vitamin my-app

# 或使用 bunx
bunx create-vitamin my-app

# 指定模板
bun create vitamin my-app --template full

# 查看所有模板
bunx create-vitamin --list
```

## 可用模板

- **basic** - 基础项目模板
- **full** - 完整功能模板
- **api** - API 服务器模板
- **minimal** - 最小化模板

## 功能

- ⏳ 交互式项目创建（计划中）
- ⏳ 多种项目模板（计划中）
- ⏳ 依赖自动安装（计划中）
- ⏳ Git 初始化（计划中）
